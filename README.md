# Simple Word Unscramble Game

A simple word unscramble game, written in Python.

## Setup

Make sure Python 3.6 and [pipenv](https://pipenv.readthedocs.io/en/latest/) is installed on your system. Then simply run

```
$ pipenv run main.py
```

to start the game.
